# Accessibility Research - For Roku App

## Description

The goal of this research assignment was to make the application accessible to all users whether they are using videos, tv shows, and audio content. 

## Download Quick Start - For Roku App Research

To download the Roku App Research segment, clone this repository and remove the `.git` folder by running the following:

```console
$ git clone https://github.com/DerrikC/Campbell_D_AccessibilityResearch.git <Campbell_D_AccessibilityResearch>
$ cd <Campbell_D_AccessibilityResearch>
```
